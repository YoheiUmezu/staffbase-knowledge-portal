# デモUI提案プロジェクト（提案担当向け）

## 何に使うか
Staffbase向けのデモUI提案を、短時間で一定品質に揃えるために使います。

## この仕組みでできること
- 提案準備の手順を共通化
- 参考資料・テンプレート・成果物サンプルを一元管理
- 関係者レビューしやすい形でアウトプットを整理

## 利用イメージ
1. `PROJECT_INSTRUCTION.md` で進め方を確認
2. `templates/` で提案骨子を作成
3. `references/` を参照してトーンを合わせる
4. `sample_deliverables/` を参考に成果物を仕上げる

## 期待できる成果
- 提案スピード向上
- UI提案品質の標準化
- 初回提案時の手戻り削減
