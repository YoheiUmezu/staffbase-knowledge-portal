# RFP回答支援（営業・提案チーム向け）

## 何に使うか
顧客から届くRFP（提案依頼書）に対して、回答の抜け漏れや根拠不足を減らし、社内でレビューしやすい形に整えるために使います。

## この仕組みでできること
- 質問ごとに「製品機能」と「運用・セキュリティ」を切り分ける
- 参照元を明確にしながら、回答の信頼性を担保する
- 不明点を「要確認」として明示し、誤回答リスクを下げる

## 利用イメージ
1. RFPの質問を分類する
2. 公式情報や社内KBから根拠を確認する
3. 制約や条件を含めて回答文を作る
4. 根拠を記録した状態で提出物に反映する

## 期待できる成果
- 回答品質の平準化
- レビュー時間の短縮
- 根拠付き回答による提案の信頼性向上
